# QRC Reader

A simple QR code reader built with JavaScript.

## Features

- Scan QR codes using your phone's camera
- Display results in real-time
- Simple and lightweight interface

## Get started

1. Clone the repository:
   ```bash
   git clone https://github.com/predragBgd/qrc-reader.git
   cd qrc-reader
   ```
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the app:

   ```bash
   npx expo start
   ```

4. Open the app:

- Scan the QR code in the terminal using the Expo Go app on your mobile device
- Or run in an emulator (iOS Simulator or Android Emulator)
